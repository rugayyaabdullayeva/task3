 1. Git nədir?
Git “version control sistemidir”.Komputerimizdə olan hər hansı projectlərimizi  internet üzərində saxlamağımızı və idarə etməyimizi təmin edən bir versiya nəzarət sistemidir .

Nə üçün gitdən istifadə etməliyik ?

Böyük projectlər üzərində işlədiyimiz zaman bir neçə nəfər eyni zamanda çalışdıgı üçün kodların birləşdirilməsi və ən son versiyanın müəyyənləşdirilməsi prosesi çox qarışıq və vaxt tələb edən prosesdir. Lakin Git vasitəsilə isə buna ehtiyac qalmır.Git vasitəsi ilə rahatlıqla hamı öz işini  yerinə yetirir və daha sonra edilmiş işlər birləşdirilir.Git ilə işləyərkən proyektlərimizi saxlamaq üçün bir çox servizlər var. Onlardan ən məhşurları və çox istifadə olunanları GitHub, GitLab və Bitbucket-dir.

 2. Github nədir?
GitHub: Private şəkildə saxlamaq istədiyimiz repository-lər üçün ödəniş tələb etsə də, Public repository-lər üçün pulsuzdur. 2008-ci ildə qurulan GitHub 2011-ci ildən etibarən Open Source proyektləri üçün ən çox istifadə olunan servisdir.
GitHub,develloperlərin kod üzərində birlikdə işləməyə, dəyişiklikləri izləməyə, kodun müxtəlif versiyalarını idarə etməyə və layihələr üzərində problemsiz şəkildə əməkdaşlıq etməyə imkan verir.

3. Gitbash nədir?

Git Bash — istifadəçilərin Git əmrlərini istifadə edə biləcəkləri bir əmr sətridir

4. “Versiyaya Nəzarət Sistemi” dedikdə nə başa düşürsünüz?

Versiyaya nəzarət sistemləri kodda edilən dəyişiklikləri izləməklə fayllara edilən dəyişiklikləri qeyd etməyə kömək edən proqram alətləri kateqoriyasıdır.
Versiyaya Nəzarət sistemi niyə bu qədər vacibdir?
Bildiyimiz kimi, proqram məhsulu bir qrup tərtibatçı tərəfindən birgə işlənib hazırlanır, onlar müxtəlif yerlərdə yerləşə bilər.
Versiyaya nəzarət sistemi, tərtibatçı qrupuna kimin etdiyi və hansı dəyişikliklərin edildiyi kimi məlumatla birlikdə mənbə koduna edilmiş bütün dəyişiklikləri səmərəli şəkildə əlaqələndirməyə və idarə etməyə (izləməyə) kömək edən bir növ proqram təminatıdır. 



5. GIT-dən istifadənin hansı faydaları var?

Bir proyekt üzərində komanda şəklində işləmək : Git vasitəsilə bir proyekt üzərində bir neşə nəfər eyni zamanda çalışa, kodları birləşdirə, ən son versiyanı müəyyənləşdirə bilərlər.
Offline istifadə edilə bilməsi: Git hər-hansı bir proyekt üzərində çalışarkən onun kopyasını local maşında saxlayaraq, internet şəbəkəsi olmadan da proyekt üzərində işləməyə imkan verir.


6. Git repozitoriyası nədir?
Layihəmizdəki bütün dəyişikliklərin GIT versiyasına nəzarət sistemi tərəfindən izlənilə və qeyd oluna bilməsi üçün bir repository yaratmalıyıq. 
Repository; Layihəmizdəki fayllara edilən dəyişikliklər və bu dəyişiklikləri kimin və nə vaxt etdiyi barədə məlumatları saxlayan verilənlər bazasıdır.
İki növ Git repositorisi var : Public Repositories , Private Repositories
